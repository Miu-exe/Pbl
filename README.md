# Garbage Truck Collector

Minimal Pygame prototype where you drive a garbage truck and collect garbage around a simple city.

Requirements
- Python 3.8+
- `pygame` (see `requirements.txt`)

Install
```bash
python3 -m pip install -r requirements.txt
```

Run
```bash
python3 main.py
```

Controls
- Arrow keys / WASD: move
- P: pause/resume
- R: restart after win
- Esc / window close: quit
